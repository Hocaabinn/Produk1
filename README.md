# 🥗 SurplusHub — Save Food, Save the Planet

**SurplusHub** adalah marketplace surplus makanan berbasis AI yang menghubungkan bisnis kuliner dengan konsumen untuk mengurangi limbah makanan secara real-time.

---

## ✨ Fitur Unggulan (AI-Powered)

* **Smart Dynamic Pricing (AI):** Penyesuaian harga otomatis berdasarkan sisa waktu operasional toko untuk memastikan *zero waste*.
* **Hyper-local Discovery:** Pencarian berbasis lokasi real-time untuk menemukan titik penyelamatan makanan terdekat.
* **CO2 Impact Dashboard:** Transparansi dampak lingkungan dengan melacak jumlah emisi karbon yang berhasil dikurangi.
* **Premium Membership:** Akses fitur eksklusif seperti *AI Image Generator* untuk mitra bisnis.

## 🛠️ Tech Stack

* **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Backend & Database:** [Supabase](https://supabase.com/) (PostgreSQL + RLS)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Deployment:** [Vercel](https://vercel.com/)
